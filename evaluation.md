# 학사 졸업논문 평가 기준

> 학사 졸업논문은 "학계에 완전히 새로운 기여를 했는가"보다,
> **명확한 연구 질문을 세우고, 적절한 근거와 방법으로 답했으며,
> 학술 논문 형식으로 완성했는가**를 중심으로 평가한다.

---

## 1. 평가 핵심 요약

좋은 학사 졸업논문은 다음 네 가지를 만족해야 한다.

1. **무엇을 묻는지 분명하다.**
   - 연구 주제와 연구 질문이 구체적이다.
   - 논문 전체가 하나의 문제의식으로 연결된다.
2. **왜 중요한지 설명한다.**
   - 선행연구와 현실적 배경을 통해 연구 필요성을 제시한다.
   - 기존 연구가 충분히 다루지 못한 지점을 밝힌다.
3. **어떻게 답했는지 투명하다.**
   - 자료, 방법, 분석 절차가 명확하다.
   - 다른 사람이 읽고 연구 과정을 이해할 수 있다.
4. **근거를 바탕으로 결론을 낸다.**
   - 주장과 자료가 논리적으로 연결된다.
   - 결과의 의미와 한계를 함께 제시한다.

## 2. 평가 항목표

| 항목 | 평가 질문 | 확인할 점 |
|---|---|---|
| **주제와 연구문제** | 무엇을 연구하는가? | 주제가 명확한가, 연구 질문이 구체적인가, 학사 수준에서 다룰 수 있는 범위인가 |
| **연구 목적과 필요성** | 왜 이 연구가 필요한가? | 문제의식이 설득력 있는가, 연구의 배경과 동기가 충분히 설명되었는가 |
| **선행연구 검토** | 기존 연구를 이해했는가? | 관련 문헌을 충분히 검토했는가, 단순 요약이 아니라 본 연구와의 관계를 설명했는가 |
| **방법론** | 어떻게 연구했는가? | 자료, 실험, 분석 기준이 연구 질문에 적절한가, 절차가 재현 가능하게 설명되었는가 |
| **분석과 논의** | 근거로부터 답을 도출했는가? | 자료 해석이 논리적인가, 결과와 주장이 잘 연결되는가, 반례나 한계를 고려했는가 |
| **독창성 및 기여** | 본인의 기여가 있는가? | 새로운 이론이 아니더라도 본인의 문제의식, 해석, 적용, 정리 방식이 드러나는가 |
| **구조와 전개** | 읽기 쉽게 구성되었는가? | 서론, 이론적 배경, 방법, 결과, 논의, 결론이 자연스럽게 이어지는가 |
| **인용과 연구윤리** | 출처를 정확히 밝혔는가? | 표절 없이 인용했는가, 참고문헌 형식이 일관적인가, 데이터와 결과를 왜곡하지 않았는가 |
| **문장과 형식** | 학술적으로 쓰였는가? | 문장이 명확한가, 구어체와 감정적 표현을 피했는가, 오탈자와 형식 오류가 적은가 |
| **발표와 질의응답** | 자기 논문을 설명할 수 있는가? | 핵심 내용을 시간 안에 설명하는가, 질문에 논문 근거로 답할 수 있는가 |

## 3. 권장 점수 배분

| 항목 | 배점 | 평가 기준 |
|---|---:|---|
| 연구 주제와 문제 설정 | 20점 | 연구 질문의 명확성, 범위의 적절성, 문제의식의 일관성 |
| 선행연구 검토 | 15점 | 관련 문헌의 충실성, 기존 연구와 본 연구의 관계 설명 |
| 방법론의 적절성 | 15점 | 자료와 방법의 적합성, 절차 설명의 투명성 |
| 분석과 논의의 깊이 | 20점 | 근거 기반 분석, 결과 해석, 한계 인식 |
| 논문 구성과 글쓰기 | 15점 | 장 구성, 문단 전개, 학술적 문체, 문장 명확성 |
| 인용, 참고문헌, 연구윤리 | 10점 | 출처 표기, 인용 형식, 표절 방지, 결과 왜곡 방지 |
| 발표 및 질의응답 | 5점 | 발표 명료성, 질문 대응, 논문 이해도 |
| **합계** | **100점** |  |

## 4. 등급별 판단 기준

| 수준 | 특징 |
|---|---|
| **우수** | 연구 질문이 명확하고, 선행연구와 방법론이 잘 연결되며, 분석 결과가 논리적으로 제시된다. 결론에서 연구의 의미와 한계를 균형 있게 설명한다. |
| **보통** | 논문 형식은 갖추었고 기본적인 분석도 수행했지만, 연구 질문이 다소 넓거나 선행연구와 분석의 연결이 약하다. 결론이 요약 수준에 머무를 수 있다. |
| **미흡** | 주제가 불명확하거나 근거 없이 주장한다. 선행연구 검토가 부족하고, 방법과 분석 절차가 충분히 설명되지 않는다. 인용이나 형식 오류가 많다. |

## 5. 한국어 학술논문 글쓰기 원칙

한국어 학술논문은 화려한 문장보다 **정확하고 객관적인 문장**이 중요하다.

### 5.1 기본 원칙

| 원칙 | 설명 | 예시 |
|---|---|---|
| **구어체를 피한다** | 말하듯 쓰지 않고 문어체로 쓴다. | `~인 것 같다` → `~로 볼 수 있다` |
| **주관적 표현을 줄인다** | 감상, 과장, 단정 표현을 피한다. | `매우 훌륭하다` → `효과가 확인되었다` |
| **근거 없이 주장하지 않는다** | 주장에는 자료, 선행연구, 분석 결과를 붙인다. | `중요하다` → `A 연구에서 중요성이 제기되었다` |
| **문장은 짧고 명확하게 쓴다** | 한 문장에 하나의 핵심만 담는다. | 긴 문장 하나보다 짧은 문장 두 개가 낫다. |
| **용어를 일관되게 쓴다** | 같은 개념은 같은 단어로 반복한다. | `환각`, `허위 생성`, `오류 생성`을 섞지 않는다. |
| **인용과 자기 주장을 구분한다** | 남의 주장과 본인의 해석을 분명히 나눈다. | `김(2020)은 ~라고 보았다. 본 연구는 이를 바탕으로 ~` |
| **1인칭 표현을 줄인다** | `나는`, `우리는`보다 연구 중심 문장을 쓴다. | `나는 분석했다` → `본 연구는 분석하였다` |
| **번역투를 피한다** | 영어식 명사구를 한국어 동사문으로 바꾼다. | `~에 대한 분석을 수행하였다` → `~을 분석하였다` |

### 5.2 자주 쓰기 좋은 학술 표현

| 상황 | 표현 |
|---|---|
| 연구 목적 제시 | `본 연구의 목적은 ~을 분석하는 데 있다.` |
| 연구 질문 제시 | `본 연구는 다음의 연구 질문을 설정하였다.` |
| 선행연구 소개 | `기존 연구에서는 ~에 주목하였다.` |
| 선행연구 한계 제시 | `그러나 기존 연구는 ~ 측면을 충분히 다루지 못하였다.` |
| 방법 설명 | `이를 위해 본 연구는 ~ 방법을 사용하였다.` |
| 분석 결과 제시 | `분석 결과, ~한 경향이 나타났다.` |
| 해석 제시 | `이는 ~을 시사한다.` |
| 한계 제시 | `본 연구는 다음과 같은 한계를 지닌다.` |
| 결론 제시 | `따라서 본 연구는 ~라고 결론 내릴 수 있다.` |

### 5.3 피해야 할 표현

| 피해야 할 표현 | 대체 표현 |
|---|---|
| `~인 것 같다` | `~로 판단된다`, `~로 볼 수 있다` |
| `엄청나게 중요하다` | `중요한 의미를 지닌다` |
| `당연히` | 삭제하거나 근거 제시 |
| `요즘 많이 쓰인다` | `최근 여러 분야에서 사용되고 있다` |
| `문제가 많다` | `다음과 같은 한계가 나타난다` |
| `좋다` / `나쁘다` | `효과적이다` / `한계가 있다` / `부적절하다` |
| `필자는 생각한다` | `본 연구는 ~라고 본다` |
| `~라고 할 수 있을 것이다` | `~라고 볼 수 있다` |

## 6. 자기 점검 체크리스트

논문 제출 전 다음 항목을 확인한다.

### 연구 내용

- [ ] 연구 질문이 한두 문장으로 명확하게 정리되는가?
- [ ] 서론에서 연구의 필요성이 설명되는가?
- [ ] 선행연구가 단순 나열이 아니라 본 연구의 배경으로 연결되는가?
- [ ] 연구 방법과 자료가 충분히 설명되는가?
- [ ] 분석 결과가 연구 질문에 답하고 있는가?
- [ ] 결론에서 연구의 의의와 한계를 함께 제시했는가?

### 글쓰기와 형식

- [ ] 같은 개념을 같은 용어로 일관되게 사용했는가?
- [ ] 한 문장이 지나치게 길지 않은가?
- [ ] 구어체, 감정적 표현, 과장 표현을 제거했는가?
- [ ] 인용한 문헌이 참고문헌 목록에 모두 포함되어 있는가?
- [ ] 참고문헌 형식이 일관적인가?
- [ ] 표, 그림, 수식 번호와 본문 설명이 맞는가?
- [ ] 오탈자와 비문을 검토했는가?

> 적절한 방법으로 분석한 뒤, 자기 힘으로 논리적인 결론을 냈는가?**
